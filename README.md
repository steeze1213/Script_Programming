# Script Programming

학교 수업 스크립트 프로그래밍을 한 학기 동안 수강하며 실습, 과제로 작성한 코드 모음입니다.  
파이썬 기초 문법부터 데이터 처리, 시각화, 머신러닝·딥러닝 기초까지 단계적으로 학습한 개인 학습 기록용 레포지토리입니다.

---

## 사용 교재

따라하며 배우는 파이썬과 데이터 과학
- 출판 연도: 2020

---

## 학습 내용 요약

### PART 1 – 파이썬 기초체력 다지기

**Chapter 01 – 파이썬 및 데이터 과학 개요**  
- 데이터 과학 개념, 개발 환경 설정, 파이썬 기초 사용

**Chapter 02 – 변수와 자료형**  
- 변수, 자료형, 입력 처리, 객체/메소드 기초

**Chapter 03 – 연산자와 수식**  
- 산술/논리/비트 연산자, 모듈 활용

**Chapter 04 – 조건문과 제어 흐름**  
- if / if-else, 조건 분기, 터틀 그래픽 활용

**Chapter 05 – 반복문**  
- for / while, break, 포매팅

**Chapter 06 – 함수와 모듈**  
- 함수 정의, 매개변수, 반환값, 재귀, 모듈화

**Chapter 07 – 리스트와 튜플**  
- 리스트 연산, 슬라이싱, 정렬, 튜플, 클래스 기초 개념

**Chapter 08 – 딕셔너리와 집합, 파일 처리**  
- 딕셔너리, 집합, 파일 입출력, 자료구조 비교

---

### PART 2 – 데이터 과학과 인공지능

**Chapter 09 – 텍스트 처리**  
- 문자열 처리, 워드클라우드, 정규표현식

**Chapter 10 – NumPy 수치 처리**  
- NumPy 배열, 인덱싱/슬라이싱, 난수, 통계

**Chapter 11 – 데이터 시각화 (matplotlib)**  
- 선 그래프, 막대 그래프, 산점도, 히스토그램, 서브플롯

**Chapter 12 – pandas 데이터 분석**  
- CSV 로딩, 필터링, 그룹핑, 병합(merge)

**Chapter 13 – seaborn 데이터 시각화**  
- 상관관계 시각화, 히트맵, 산점도, 페어플롯

**Chapter 14 – 머신러닝 기초**  
- 선형 회귀, k-NN 분류, 사이킷런 활용

**Chapter 15 – 딥러닝 기초**  
- 퍼셉트론, 신경망, 텐서플로, MNIST 이미지 분류
